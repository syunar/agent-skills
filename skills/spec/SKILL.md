# Spec

Treat invocation arguments as initial task context.

Load and follow `grill-with-docs` first. Resolve the design with the user and
preserve its confirmation boundary: do not enact the work before shared
understanding is confirmed.

After confirmation, load and follow `to-spec` to synthesize the agreed context
into `docs/specs/YYYY-MM-DD-<feature-slug>.md`.

Do not modify product code.
